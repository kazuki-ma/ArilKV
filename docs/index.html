Name

An aril is the seed wrapped in red.

Garnet traces back to granatus, the seeded fruit. ArilKV keeps that lineage but moves closer to the data model: many small, durable units gathered into a fast in-memory structure.

Architecture

A small surface with serious internals.

Owner-thread execution

Mutation paths are serialized where locality matters, keeping routing explicit instead of hiding contention behind broad locks.

Redis-compatible protocol

RESP command handling is tested through Rust regressions and external compatibility probes as the command surface grows.

Evidence-first performance

Benchmark scripts capture parameters, artifacts, integrity checks, and caveats so performance work can be repeated.

Formal-model hooks

Critical concurrency paths are tracked against TLA+ models where the implementation needs stronger invariants.
