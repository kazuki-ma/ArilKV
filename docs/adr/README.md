# ADR Index

This directory stores Architecture Decision Records (ADRs) for decisions that
we intentionally keep, defer, or reject.

Conventions:

- Filename: `NNNN-short-title.md`
- Required header fields:
  - `Status`
  - `Date`
  - `Owners`
- Each ADR should include:
  - `Context`
  - `Decision`
  - `Consequences`
  - `Revisit Triggers` (explicit conditions to reopen the decision)
